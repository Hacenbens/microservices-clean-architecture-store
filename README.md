# Microservices Clean Architecture Store

This project is a learning-oriented microservices system built with:

- .NET
- Clean Architecture
- CQRS
- Docker
- Service Discovery
- API Gateway

## Services
- User Service
- Product Service
- Order Service
- API Gateway

## Architecture
Each service follows Clean Architecture:
- Domain
- Application
- Infrastructure
- API
